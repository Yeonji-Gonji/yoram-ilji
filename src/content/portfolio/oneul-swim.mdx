생활 속 불편을 직접 풀어보려고 만든 개인 프로젝트입니다.
"오늘 이 시간에 자유수영을 할 수 있는 수영장이 어디지?"라는 질문을, 권역별 지도와 요금·시간 정보로 한 화면에서 답하게 했습니다.

## 만들면서 신경 쓴 것

- **데이터부터 가볍게**: 정적 JSON(`pools.json`)으로 시작해 MVP를 빠르게 세우고, 외부 의존을 최소화했습니다.
- **지도 폴백**: Kakao Maps 키가 없거나 도메인 미등록(401)이어도 앱이 죽지 않도록 **권역 핀 버튼 폴백**을 넣어, 키 없이도 빌드·구동되게 했습니다.
- **PWA**: 모바일에서 앱처럼 설치·사용할 수 있도록 PWA로 구성했습니다.
- **자동화**: GitHub Actions로 `lint → typecheck → build` 게이트를 걸고, Vercel 연동으로 `main` 푸시 시 자동 배포 + PR마다 프리뷰 URL이 뜨게 했습니다.

혼자 하는 프로젝트라도 CI/CD를 먼저 깔아두면 손이 덜 간다는 걸 다시 확인한 작업입니다.
